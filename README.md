# music
> 这是一款音乐app,接口来自qq音乐

## 技术栈
* vue
* vuex
* axios
* better-scroll
* node

## 演示
[项目演示地址](http://94.191.23.56:83/index.html#/recommend)

## Usage
```
# Clone this repository
git clone https://github.com/DFairy/music.git

# Go into the repository
cd music

# Install dependencies
npm install

# Run the app
node prod.server.js
npm run serve
```
